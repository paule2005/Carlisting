# New Color Scheme - Black & Orange Theme

Inspired by Pics Enhancer design

## Color Palette

### Primary Colors
- **Background**: `#000000` (Pure Black)
- **Surface**: `#1a1a1a` (Dark Gray)
- **Card Background**: `#0d0d0d` (Slightly lighter black)

### Accent Colors
- **Primary Orange**: `#ff6b35` (Coral Orange)
- **Secondary Orange**: `#ff8c42` (Light Orange)
- **Hover Orange**: `#ff4500` (Deep Orange)

### Text Colors
- **Primary Text**: `#ffffff` (White)
- **Secondary Text**: `#b3b3b3` (Light Gray)
- **Muted Text**: `#808080` (Gray)

### Status Colors
- **Success**: `#10b981` (Green)
- **Error**: `#ef4444` (Red)
- **Warning**: `#f59e0b` (Amber)

## Gradients
- **Primary Gradient**: `linear-gradient(135deg, #ff6b35 0%, #ff8c42 100%)`
- **Dark Gradient**: `linear-gradient(135deg, #1a1a1a 0%, #0d0d0d 100%)`
- **Overlay Gradient**: `linear-gradient(to top, rgba(0, 0, 0, 0.95) 0%, rgba(0, 0, 0, 0.5) 70%, transparent 100%)`
